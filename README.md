# SharedBrain

Shared memory layer for three AI agents: **Cascade**, **Mutik**, and **ChatGPT**.

- **Cascade** (Windsurf/Claude) and **Mutik** (OpenClaw) connect via MCP server
- **ChatGPT** (Web Business) syncs via this GitHub repo + Issue #7
